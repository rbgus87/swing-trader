## 인증 (Authentication)

- [ ] 비밀번호 해싱 (bcrypt/argon2, 라운드 12+)
- [ ] JWT 시크릿 충분한 엔트로피 (256비트+)
- [ ] 토큰 만료 설정 (access: 15분, refresh: 7일)
- [ ] 리프레시 토큰 로테이션 구현
- [ ] 로그인 실패 횟수 제한 (5회/15분)
- [ ] 세션 무효화 로직 (로그아웃, 비밀번호 변경 시)

## 인가 (Authorization)

- [ ] 모든 엔드포인트에 인가 검증 적용
- [ ] RBAC/ABAC 정책 일관성 확인
- [ ] 수직 권한 상승 방지 (일반 사용자 → 관리자)
- [ ] 수평 권한 상승 방지 (사용자 A → 사용자 B 리소스)
- [ ] 관리자 엔드포인트 분리 및 추가 인증

## 입력 검증 (Input Validation)

- [ ] 서버 사이드 입력 검증 (클라이언트 검증에 의존 금지)
- [ ] SQL Injection 방지 (파라미터 바인딩, ORM 사용)
- [ ] XSS 방지 (출력 인코딩, CSP 헤더)
- [ ] CSRF 방지 (토큰 또는 SameSite 쿠키)
- [ ] 파일 업로드 검증 (타입, 크기, 확장자)
- [ ] URL 리다이렉트 검증 (오픈 리다이렉트 방지)
